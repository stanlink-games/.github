## 🎮 Ma3

**Ma3** is an open-world game set in the streets of Nairobi, Kenya. 
Drive, customize, hustle. From beat-up ma3 to legendary nganya.

- 🌍 Real Nairobi — real routes, real culture, real chaos
- 🚌 Deep vehicle customization — paint, sound, suspension, soul
- 🌃 Day/night/rain — the city breathes
- 🎵 Kenyan radio — gengetone, benga, reggae, your mix

**Status:** Early development. Vertical slice in progress.
